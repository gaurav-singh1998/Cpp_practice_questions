(a+b)%m =((a%m) + (b%m))%m

(a-b)%m =((a%m) - (b%m) + m)%m

(a*b)%m =((a%m) * (b%m))%m

(a/b)%m = ((a%m) * (inv(b)%m))%m
inv(b) = multiplicative modulo inverse(b)

if a number 'a' has multiplicative inverse 'b' than a*b = 1;

if a number 'a' has multiplicative modulo inverse 'b' than (a*b)%m = 1 thus in this case
'b' is the multiplicative modulo inverse of a w.r.t m;
